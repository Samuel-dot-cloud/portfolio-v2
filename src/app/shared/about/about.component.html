Let me introduce myself

My journey started back in 2019, when I started learning basic web design, how to implement agile methodologies, and how to sketch wireframes for project ideation. By undertaking courses at a bootcamp, I gained proficiency in HTML, CSS, JavaScript and Jquery, which enabled me to create cool websites.

I then progressed further to learn Angular, a web framework for creating fully scalable websites and progressive web applications. I proceeded to get my hands dirty with Spark, a web framework for creating websites and APIs' with Java as the main programming language. With Java under my belt, I also ventured into the Android platform, which has enabled me create fully functional and scalable mobile applications that work on normal Android phones and tablets.

As of now, I'm still a learner at heart, sharpening my craft, while researching and taking note of the latest trends in the tech scene. I prefer a hands-on, practical approach with clients, which enables me to come up with the best and most applicable solution to their business and personal needs.
